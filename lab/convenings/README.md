## Recording an Exploration

Create a file in this directory when starting a new exploration:

```markdown
# [Exploration Name]

**Type**: Working Group / Hackathon / Research Circle / etc.
**Status**: Active / Completed / Archived
**Started**: [Date]
**Ended**: [Date or "ongoing"]

## Question

What question is this exploration trying to answer?

## Participants

- [Name/handle] - [role/expertise]

## Outputs

- [Link to spec/report/prototype]

## Summary

What happened and what was learned.
```

Not every exploration needs detailed documentation. Use judgment.
